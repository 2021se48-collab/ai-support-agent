# 🤖 AI Support Agent

An intelligent customer support agent built with FastAPI, OpenAI, and Python.

## ✨ Features
- REST API with FastAPI
- AI-powered chat endpoint using Groq LLaMA model
- Automated response system
- Web chat UI
- Deployed on Render

## 🛠️ Tech Stack
- **Backend:** Python, FastAPI
- **AI Model:** Groq LLaMA
- **Frontend:** HTML/CSS/JS
- **Deployment:** Render

## 🚀 Getting Started

### 1. Clone the repo
git clone https://github.com/2021se48-collab/ai-support-agent.git
cd ai-support-agent

### 2. Install dependencies
pip install -r requirements.txt

### 3. Set up environment variables
cp .env.example .env
# Add your API keys in .env

### 4. Run the app
uvicorn app.main:app --reload
